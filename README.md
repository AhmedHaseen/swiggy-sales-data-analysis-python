# Swiggy Sales Data Analysis

## Project Overview

This project analyzes Swiggy sales data using Python to uncover sales trends, customer ordering patterns, and regional performance insights.

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Jupyter Notebook
