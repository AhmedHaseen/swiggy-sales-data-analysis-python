# Swiggy Sales Data Analysis

## Project Overview

This project analyzes Swiggy sales data using Python to uncover sales trends, customer ordering patterns, and regional performance insights. The main analysis is done in the Jupyter notebook, with charts saved in the `Images` folder.

## What This Project Covers

- Total revenue and average order value
- Monthly and daily sales trends
- Revenue split by veg and non-veg categories
- Top states by revenue
- Top dishes by total revenue

## Files

- `Swiggy Sales Analysis.ipynb` - main analysis notebook
- `swiggy_data.xlsx` - source dataset
- `Images/` - saved chart images from the analysis

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Jupyter Notebook

## How to Run

1. Open `Swiggy Sales Analysis.ipynb` in Jupyter Notebook or VS Code.
2. Make sure the dataset path points to `swiggy_data.xlsx`.
3. Run the cells from top to bottom.

## Notes

- The project is intentionally simple and is suitable for a basic portfolio or learning project.
- If you want to share or reuse it later, you can add a `requirements.txt` file with the Python packages used.
